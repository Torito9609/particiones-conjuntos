# Visualizador de Particiones de Conjuntos

Proyecto en Python para generar y visualizar todas las particiones de un conjunto,
con validación matemática mediante números de Bell y Stirling.

## Referencias

Este proyecto adapta y extiende los algoritmos descritos en:

> Giorgos Stamatelatos & Pavlos S. Efraimidis.  
> *Lexicographic Enumeration of Set Partitions* (2021).  
> [arXiv:2105.07472](https://arxiv.org/abs/2105.07472).  
> Repositorio original: [github.com/gstamatelat/partitions-enumeration](https://github.com/gstamatelat/partitions-enumeration)

Ver el archivo [CITATION.cff](./CITATION.cff) para la referencia completa.
